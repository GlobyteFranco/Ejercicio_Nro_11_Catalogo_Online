# Ejercicio_Nro_11_Catalogo_Online
Ejercicio Numero 11 de la guia practica de Java
# Consigna
El departamento de marketing de una empresa desea un software que diseñe su catálogo
online de venta estos son preparados de forma mensual y constan de artículos que
pueden ser, solo texto, con imágenes, o multimedia con vídeos. Diseñe un software que
permita organizar los catálogos
